## Project Overview
This task implements a basic IP tracking system in a Django application.  
The goal is to log the **IP address**, **request path**, and **timestamp** of every incoming HTTP request using Django middleware.

This functionality is foundational for security monitoring, auditing, and future features such as rate limiting and IP blacklisting.

---

## Objective
- Log every incoming request
- Store:
  - Client IP address
  - Request path
  - Timestamp of the request
